## Usage

### Partager avec un client
```
1. Ouvrir index.html dans navigateur
2. Cliquer sur le livrable voulu
3. Imprimer ou partager par email
```

### Mettre à jour un deliverable
1. Éditer le fichier HTML en local
2. Tester dans navigateur
3. Committer si changement majeur
4. Dater le footer avec la dernière MAJ

### Ajouter un nouveau deliverable
1. Créer fichier HTML dans ce dossier
2. Ajouter une card dans index.html
3. Dater + tester print/mobile
