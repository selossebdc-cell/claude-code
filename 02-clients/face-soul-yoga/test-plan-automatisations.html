🚨 Plan de Rollback : Si quelque chose casse en prod, comment revenir en arrière rapidement.
⛔ Rollback — Retrouver la Situation
1
Si Emails ne s'envoient plus
Action Rapide
Aller Brevo → Automations → vérifier tous les interrupteurs (ON ou OFF ?)
Hypothèses à tester :
1️⃣ Une automation OFF ? Cliquer le bouton vert "Activer"
2️⃣ n8n workflow DOWN ? Aller n8n → vérifier les logs d'erreur
3️⃣ API credentials invalide ? Tester une notification test dans Brevo
4️⃣ Contact pas dans la bonne automation ? Vérifier les attributs (INTERET, STATUT_MEMBRE, OFFRE)
⏱️ Temps résolution : 2-5 min
2
Si Achat Stripe ne crée pas de STATUT_MEMBRE=active
Action Rapide
Stripe → Settings → Webhooks → voir si y'a des "Failed" deliveries
Hypothèses :
1️⃣ Webhook OFF ? L'activer dans Stripe
2️⃣ Webhook URL mauvaise ou changée ? Mettre à jour l'endpoint n8n
3️⃣ n8n Workflow 01 OFF ? L'activer
4️⃣ Brevo API credentials dans n8n invalides ? Les regénérer et les re-configurer
⏱️ Temps résolution : 5-10 min
3
Si Contact ne reçoit pas d'invitation Circle
Action Rapide
n8n → Workflow 04 → voir les logs de la dernière exécution
Hypothèses :
1️⃣ Workflow 04 OFF ? L'activer
2️⃣ Circle API Token expiré ? Regénérer et re-mettre dans n8n
3️⃣ Erreur "Group ID invalide" ? Vérifier les IDs Circle (1035229 pour FSY, 1025924 pour MTM)
4️⃣ Contact pas dans Brevo avec bon OFFRE ? Vérifier l'attribut OFFRE
⏱️ Temps résolution : 3-8 min
4
Si Contact reçoit des emails en double (ou pas du tout)
Action Rapide
Brevo → Contact → voir "Email Activity" ou "Automation History"
Hypothèses :
1️⃣ Email dans liste d'exclusion/blocage ? Vérifier Brevo → Suppressed contacts
2️⃣ Automation lancée 2x (doublon contact) ? Faire un merge dans Brevo
3️⃣ Contact ne match pas les conditions (INTERET ≠ yoga) ? Ajouter manuellement l'attribut
⏱️ Temps résolution : 3-5 min
5
Plan Contingency : Revenir à la Situation Avant
Scénario Catastrophe
Quelque chose a vraiment cassé, on veut revenir à l'état de AVANT les tests
Étapes :
1. Désactiver les workflows n8n :
  n8n → Workflow 01, 04, 05, 07, 08 → toggle "Active" OFF

2. Désactiver les automations Brevo :
  Brevo → Automations → cliquer sur chaque → bouton "Pause" ou OFF

3. Désactiver le webhook Stripe (tempo) :
  Stripe → Settings → Webhooks → trouver l'endpoint → cliquer sur le toggle OFF

4. Supprimer les contacts de test :
  Brevo → Contacts → chercher test-fsy@, test-mtm@ → supprimer

5. Annuler les subscriptions de test dans Stripe :
  Stripe → Subscriptions → chercher les subscriptions test → Cancel
⏱️ Temps total : 10 min
📋 Checklist Rollback
✅ Après Rollback : Aucun workflow ne tourne, aucun email n'est envoyé, Stripe ne parle pas à n8n. Vous êtes à l'état "avant tests". Vous pouvez redémarrer tranquillement après correction.
