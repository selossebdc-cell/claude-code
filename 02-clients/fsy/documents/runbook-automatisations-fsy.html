🎯 Runbook Automatisations FSY

Parcours client complet — Pour déléguer à une personne sans expérience

Objectif : Une personne reçoit une vente (Stripe), l'email s'envoie automatiquement, elle rentre dans Circle, reçoit les tutoriels, et on la re-nurture si elle churn. Zéro action manuelle après Stripe.

Diagnostic & Dépannage — Si quelque chose ne marche pas

🔴 Contact n'a pas reçu d'email (Brevo sequence)
❓ Je viens de créer un contact, il n'a pas reçu l'email de nurture
Check 1 : L'automation Brevo est-elle activée ? (Brevo → Automations → voir l'interrupteur ON)
Check 2 : Le contact a-t-il l'attribut clé ? (ex. INTERET=yoga pour nurture yoga)
Check 3 : Brevo historique → voir si l'email a été envoyé (statut "sent" ou "failed")
Check 4 : Spam ? Demander au contact de vérifier ses spams
Fix : Aller dans Brevo, chercher le contact, activer manuellement l'automation
❓ Contact a acheté sur Stripe, mais Brevo n'a pas été mis à jour (STATUT_MEMBRE reste vide)
🚨 Ceci est le blocage #1 : Stripe webhook ne parle pas à n8n
Check 1 : Stripe → Settings → Webhooks → voir si y'a un endpoint n8n (URL qui ressemble à https://n8n.example.com/webhook/abc123)
Check 2 : Si endpoint existe, vérifier les "Events" : doit avoir charge.succeeded
Check 3 : Cliquer sur l'endpoint → "Event Deliveries" → voir si Stripe a essayé d'envoyer le webhook
Check 4 : Si "failed", cliquer sur la livraison → voir le message d'erreur (timeout ? 404 ?)
Fix : Demander à Laurie de copier l'URL n8n Workflow 01 et de la mettre dans Stripe
❓ Contact a acheté, STATUT_MEMBRE=active est bon, mais le contact n'a pas été invité à Circle
Check 1 : Vérifier que Workflow 04 (Stripe/Circle → Invite) existe et est activé dans n8n
Check 2 : Vérifier que l'API Circle token est bon dans les credentials n8n
Check 3 : n8n → Workflow 04 → voir les exécutions (logs) : y'a-t-il une erreur ?
Check 4 : Si erreur "Invalid Community ID" ou "Invalid Group ID", les IDs Circle sont mauvaises
Fix : Copier les vrais IDs depuis Circle → Settings et les mettre à jour dans Workflow 04
❓ Contact n'a pas reçu les emails Onboarding (après achat)
Check 1 : Brevo automation "Onboarding FSY Studio" est-elle activée ? (interrupteur ON)
Check 2 : Contact a-t-il STATUT_MEMBRE=active ET OFFRE=fsy_studio ?
Check 3 : Brevo → Automations → chercher l'automation → voir "Contacts" ou "History"
Fix : Aller dans Brevo, activer l'automation si elle est OFF
