# Template compte-rendu de séance — Format Notion

Ce template est basé sur les vrais CR de Catherine pour ses clients. Chaque CR est une page Notion créée sous le "Meeting Agendas" du dashboard client.

## Structure exacte à reproduire

Le CR complet va dans le **CONTENU de la page** Meeting Agendas (pas juste les propriétés).

### Éléments du CR dans l'ordre :

```markdown
**🎥 Enregistrement :** [Lien Fathom]

## 🎯 SESSION [N] - [Titre]

### ✅ CÉLÉBRATIONS
- [Ce qui a fonctionné]
- Chaque célébration avec contexte et emoji 🔥 si gros win
- **À reproduire** : [bonne pratique à ancrer]

### 🔄 BLOCAGES IDENTIFIÉS
- **[Sujet]** : Description du blocage
- **[Sujet]** : Description du blocage

---

## 🛠️ DÉCISIONS PRISES

### 1️⃣ [Décision]
- Détails et justification
- Prochaines étapes liées

### 2️⃣ [Décision]
- Détails de la décision

---

## 🎯 ACTIONS CLIENT (Semaine du [dates])

### ⚡ Priorité 1 : [Titre] 🔴
- [ ] Action concrète
- [ ] Action concrète

### ⚡ Priorité 2 : [Titre] 🟠
- [ ] Action concrète
- [ ] Action concrète

---

## 📝 ACTIONS CATHERINE
- [ ] [Listées ici pour référence, créées dans base centrale]

---

## 🧘 MINDSET & INSIGHTS
[Prises de conscience]

### **Ce qui fonctionne** ✅
- [Bonnes pratiques validées]

### **Prises de conscience** 💡
- "[Citation directe du client]" (contexte)
- [Insight extrait de la conversation]

---

## 📅 PROCHAINE SESSION
**Date :** [Date et heure]
**Focus :** [Thème prévu]
```

## Codes couleur priorités
- 🔴 Priorité 1 (urgent et important)
- 🟠 Priorité 2 (important, pas urgent)
- 🟡 Priorité 3 (à faire quand possible)

## Statuts projets
- DÉMARRAGE IMMINENT
- EN COURS
- EN ATTENTE
- À AUTOMATISER
- TERMINÉ

## Règles de rédaction
- Utiliser les MOTS EXACTS du client entre guillemets pour les insights
- Les actions sont toujours avec des checkboxes (- [ ])
- Chaque action est concrète et actionnable (pas de "réfléchir à...")
- Les célébrations d'abord — toujours commencer positif
- Le contexte perso est important — il humanise le suivi et permet d'adapter
- Liens vers les pages Notion existantes quand pertinent
