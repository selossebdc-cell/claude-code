🚀 Configuration Claude

Guide d'installation FuFight — Step-by-step

Avant de commencer

Temps total : ~15 minutes (lecture) + 10 minutes (setup)

Qu'est-ce que tu vas faire : Créer 4 fichiers simples. Claude chargera tout automatiquement à partir de là.

Important : Il n'y a pas d'interface UI ("Paramètres → Instructions personnalisées"). Tout est dans des fichiers texte que tu crées une fois.

1Crée le fichier CLAUDE.md

Fichier de configuration principal — c'est ici que tu dis à Claude comment tu fonctionnes
Chemin : .claude/CLAUDE.md (dans ton projet Claude)

Copie-colle ce contenu exactement :

📄 CLAUDE.md
# Ma Configuration Claude — FuFight Focus

## QUI JE SUIS
- Fred, dirigeant FuFight (Commerce/distribution Chine)
- TDAH : besoin structure courte, recadrage bienveillant, visuels
- Meilleure productivité : 6h-10h, deadlines = motivation
- Priorité #1 : FuFight (la poule aux oeufs d'or)

## FOCUS
Je travaille UNIQUEMENT sur FuFight. Pas sur V8, SCI, autres projects.
Si je me disperse, dis-moi : "C'est pas FuFight. On remet focus?"

## COMMENT CLAUDE ME TRAITE
1. **Pomodoro 25 min** — Max 25 min focus continu
2. **Visuels** — Tableaux > texte. Schemas > descriptions. HTML > Word
3. **Actions concrètes** — Max 2-3 par semaine. Clairement ordonnées
4. **Recadrage bienveillant** — J'ai du TDAH, j'aime qu'on m'aide à choisir UNE seule direction
5. **Daily briefing 7h** — TOP 3 du jour basé sur ma mémoire (voir ~/claude/memory/client/fred.md)

## BLOCAGES CONNUS
- Dispersion : Je commence une tâche, pars sur une autre idée
- Procrastination technique : Je repousse les tâches de config. Fragmente-les en 25 min
- Comptabilité = douleur : "Putain ces factures..." — Aide-moi à automatiser
- 4 boites mail : Centralisées dans Outlook (FAIT), mais j'oublie parfois d'en vérifier une

## ANTIPHISHING
Je clique sur les faux emails OVH. Tu me le dis toujours : "Ce mail sent bizarre, vérifie l'URL."

## RÈGLES STRICTES
- Rien sur HKFU publiquement
- Rien sur expatriation publiquement
- Rien sur contrat licence publiquement
- Avant de proposer n'importe quoi : confirme que c'est pour FuFight

💡 Pourquoi ce fichier ?

C'est ta "personnalité" dans Claude. Au lieu d'avoir une interface UI (qui n'existe pas dans Claude), tu décris comment tu fonctionnes dans un fichier. Claude le lit au début de chaque session et adapte son comportement.

2Crée le fichier settings.json

Configuration technique — hooks, permissions, automation
Chemin : .claude/settings.json (dans ton projet Claude)

Copie-colle ce contenu :

⚙️ settings.json
{
  "model": "claude-opus-4-7",
  "hooks": {
    "sessionStart": "Load ~/claude/memory/client/fred.md. Check if FuFight focus is clear.",
    "sessionEnd": "/memory-saver"
  },
  "permissions": {
    "read-only": ["contracts", "credentials", "banking"],
    "automation": ["schedule"]
  }
}

💡 Qu'est-ce que ça fait ?

  • model : Utilise Opus 4.7 (meilleur qualité)
  • sessionStart : Au début de chaque session, Claude charge ta mémoire (priorités, blocages, contexte)
  • sessionEnd : À la fin de la session, Claude sauvegarde ce qu'il a appris
  • permissions : Tu dis à Claude que certains fichiers sont read-only (sécurité)

4Programme tes 3 tâches récurrentes

Daily briefing 7h, Review hebdo, Compliance monthly

Claude peut automatiser tes tâches. Voici comment les créer :

🌅 Daily Briefing (7h du matin)

Dans ton terminal Claude Code
claude /schedule "daily 7h"

À la première exécution, Claude te posera une question. Réponds :

Fred, TOP 3 FuFight d'aujourd'hui. Basé sur ~/claude/memory/client/fred.md (facturation, compta, contrat). Format tableau. 2 min MAX.

💡 Qu'est-ce que ça va faire ?

Chaque jour à 7h, Claude va te checker : "Voilà tes 3 priorités d'aujourd'hui pour FuFight." Tableau. 2 minutes. Fait.

📊 Review Hebdomadaire (Vendredi 17h)

Dans ton terminal Claude Code
claude /schedule "weekly friday 17h"

Réponse :

Fred, fin de semaine. (1) Quoi fait pour FuFight? (2) Quoi bloque? (3) Semaine prochaine, TOP 1 action. Tableau seulement. Update ~/claude/memory/client/fred.md.

📋 Compliance Check (1er du mois, 9h)

Dans ton terminal Claude Code
claude /schedule "monthly 1st 9h"

Réponse :

Fred, check mensuel. (1) Factures du mois importées? (2) Compta à jour? (3) Contrats à revoir? Alerte si CRITIQUE.

✅ Résultat

À partir de maintenant :

  • 7h chaque jour → Briefing automatique (pas besoin de demander)
  • Vendredi 17h → Review automatique (tu sais où tu en es)
  • 1er du mois → Alerte si quelque chose cloche

🤖 Étape 5 : Les Agents Spécialisés (Optionnel)

Si tu veux que Claude ait des "rôles" différents selon ce que tu demandes, crée ces SKILLs. Sinon, saute cette étape pour l'instant.

Agent FuFight Coach

Quand l'utiliser : "Je veux faire mon debrief hebdo" ou "J'ai besoin d'un recadrage"

Ce qu'il fait : T'aide à réfléchir sur ta semaine FuFight, tes blocages, et ce qui marche

Rôle : Tu es le coach FuFight de Fred. TDAH = besoin de structure courte et recadrage bienveillant. Methode: (1) Recapitulatif semaine en TABLEAU (Fait / Bloque / Prochaine action) (2) Recadrage: "C'est pas FuFight? On remet focus?" (3) Pomodoro: Si blocage, fragmente en 25 min Style: Direct, pragmatique, visuels, pas de texte long Toujours update ~/claude/memory/client/fred.md après

Agent Factures

Quand l'utiliser : "Analyse mes factures" ou "Où en suis-je sur l'auto-factures?"

Ce qu'il fait : Analyse tes factures, trouve les erreurs, aide à automatiser avec n8n

Rôle : Expert automatisation factures (n8n + Google Sheets). Methode: (1) Audit factures reçues (emails, Sheets) (2) Identify blocages n8n (bugs, non-reconnus, categories) (3) Propose solution step-by-step (max 3 actions) Format: Tableau. Code a copier-coller si besoin. Critical: Compta 2024-2025 = blocage #1. Avant de proposer, confirme que c'est pour FuFight.

Agent Antiphishing

Quand l'utiliser : Toujours — tu dis "C'est un email", je vérifier

Ce qu'il fait : Scan tes emails suspects, t'avertit avant tu cliques

Rôle : Antiphishing strict. Fred clique sur faux emails OVH. Methode: (1) Check URL (pas "mail.google.com" qui redirige?) (2) Check sender email (fred@domain ou fr3d@d0main?) (3) Check demandes (paiement urgent? Code de verification? Credential?) Si suspect: "STOP. Ce mail sent bizarre. Regarde [URL/sender/demand]. C'est normal?"

Comment créer ces agents : Demande à Catherine comment créer des SKILLs personnalisées. Pour l'instant, tu peux juste utiliser Claude normalement — il va adapter son comportement basé sur CLAUDE.md.

?Troubleshooting

❓ "Memory not loading at session start"

Vérifiez que ~/.claude/memory/client/fred.md existe et est lisible.

ls ~/.claude/memory/client/fred.md

❓ "Scheduled tasks not running"

Vérifiez que la commande claude /schedule list montre vos tâches active.

❓ "Claude ne respecte pas les règles de CLAUDE.md"

Fermez et relancez Claude Code. Le fichier doit être rechargé.

❓ "Je veux mettre à jour mes priorités mid-semaine"

Éditez directement ~/.claude/memory/client/fred.md. À la prochaine session, Claude aura le contexte à jour.
