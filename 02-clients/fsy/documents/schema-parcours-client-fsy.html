🎯 Parcours Client FSY — Comment ça fonctionne

1
📱 Prospect voit une vidéo, clique sur un lien
Source
YouTube, Instagram, Email, Google
Il clique sur
Un lien avec UTM
Exemple : Un prospect clique sur une vidéo YouTube → lien : https://csbusiness.fr/yoga?utm_source=youtube&utm_medium=video&utm_campaign=mars_2026
Brevo voit d'où il vient (YouTube).
⬇️
2
🧠 Prospect remplit un quiz (Manychat)
Outil
Manychat (Bot Telegram)
Il répond
"Ça m'intéresse : Yoga"
Qu'est-ce qui se passe : Manychat envoie ses réponses (email + intérêt) à n8n Workflow 05
⬇️
3
🤖 Contact créé dans Brevo (n8n Workflow 05)
Dans Brevo
Contact créé avec attributs
Email: john@example.com INTERET: yoga SOURCE: youtube STATUT_MEMBRE: (vide, pas encore acheté)
Qu'est-ce qui se passe : Brevo reçoit le contact. Comme INTERET = yoga, l'automation Brevo "Nurture Yoga" s'active automatiquement.
⬇️
4
📧 Reçoit des emails de présentation (Brevo Automation)
Email #1 (J+0)
"Bienvenue Yoga!"
Email #2 (J+2)
"Voici comment ça marche"
Email #3 (J+5)
"Achète pour 17€/mois"
Qu'est-ce qui se passe : Brevo Automation "Nurture Yoga" envoie des emails automatiquement pour présenter le produit.
⬇️
5
💳 Prospect achète (Stripe)
Il clique sur
Bouton "Acheter 17€"
Il paie avec
Carte bancaire (Stripe Checkout)
Qu'est-ce qui se passe : Stripe enregistre la charge. Stripe envoie une notification à n8n.
⬇️
6
🔄 Stripe parle à n8n (Webhook)
Stripe dit à n8n
"Charge réussie!"
n8n Workflow
Workflow 01
Stripe webhook reçu: charge_id: ch_xxx amount: 17.00 EUR customer_email: john@example.com product: FSY_Studio status: succeeded ✓
C'est LE moment clé : Stripe dit à n8n Workflow 01 "Il a acheté FSY Studio". n8n reçoit le message et met à jour Brevo.
⬇️
7
✅ Brevo mis à jour (n8n Workflow 01)
Attributs mis à jour
Voir ci-dessous
Email: john@example.com INTERET: yoga SOURCE: youtube STATUT_MEMBRE: active ✓ (NOUVEAU!) OFFRE: fsy_studio ✓ (NOUVEAU!) DATE_ACHAT: 2026-04-20 ✓ (NOUVEAU!)
Qu'est-ce qui se passe : Brevo voit STATUT_MEMBRE = active. Automatiquement, l'automation "Onboarding FSY Studio" s'active.
⬇️
8
📧 Reçoit emails d'accueil (Brevo Automation)
Email #1 (J+0)
"Bienvenue client!"
Email #2 (J+3)
"Voici Circle + accès vidéos"
Email #3 (J+7)
"Guide complet + FAQ"
Qu'est-ce qui se passe : Brevo Automation "Onboarding FSY" envoie des emails pour l'accueillir et lui expliquer comment utiliser Circle.
⬇️
9
👥 Invité dans Circle (n8n Workflow 04)
n8n Workflow 04
Invite dans Circle
Group Circle
FSY Studio (550+ vidéos)
Qu'est-ce qui se passe : n8n Workflow 04 reçoit l'email du contact et l'invite automatiquement dans Circle. Il peut maintenant voir toutes les vidéos.
⬇️
