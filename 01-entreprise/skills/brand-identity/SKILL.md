# Brand Identity — CS Consulting Stratégique / Catherine Selosse — Consulting Stratégique

Ce skill contient toute l'identité visuelle de Catherine Selosse — Consulting Stratégique. Il sert de référence pour tous les documents, présentations, emails, et supports visuels.

Avant de commencer, lis la charte complète :
- `references/charte-graphique.md` — couleurs, typographies, logos, règles d'utilisation

## Quand ce skill se déclenche

- Catherine demande quel logo / quelle couleur utiliser
- Un autre skill (proposal-generator, invoice-generator) a besoin de l'identité visuelle
- Catherine crée un document, une présentation, un post, un email branded
- Catherine demande de vérifier la cohérence visuelle d'un support

## Règles d'utilisation

1. Toujours utiliser le logo "Consulting Stratégique" pour les documents business (pas "RH Consulting")
2. Zone de protection : laisser un espace autour du logo
3. Ne pas déformer : toujours conserver les proportions
4. Privilégier les combinaisons prévues (Terracotta sur Lin, Blanc sur Terracotta, etc.)
5. Jamais de couleurs hors palette
