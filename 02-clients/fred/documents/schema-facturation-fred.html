2b Tuto — Configurer un filtre Gmail (pas à pas)

Ce tuto s'applique aux boîtes Gmail de Fred (Gmail perso + fred@v8 + v8@v8). Pour les boîtes OVH (fu@fusolution.fr, contact@v8), c'est une autre procédure (voir plus bas).

Ouvrir Gmail dans le navigateur

Se connecter à la boîte mail à configurer (ex : fred@v8-equipment.com) dans Firefox.
Important : utiliser le bon conteneur Firefox pour la bonne boîte.

Aller dans les Paramètres

Cliquer sur la roue dentée ⚙️ en haut à droite, puis "Voir tous les paramètres".

⚙️ › Voir tous les paramètres

Aller dans l'onglet "Filtres et adresses bloquées"

Dans la barre d'onglets en haut de la page Paramètres, cliquer sur "Filtres et adresses bloquées".
Tu verras la liste des filtres existants (s'il y en a).

Paramètres › Filtres et adresses bloquées

Créer un nouveau filtre

En bas de la liste des filtres, cliquer sur "Créer un filtre".
Un formulaire apparaît avec plusieurs champs.

Remplir le champ "Contient les mots"

Laisser tous les champs vides sauf le champ "Contient les mots".
Coller la requête complète :

{facture factures avoir avoirs "note d'honoraires" relevé acompte échéance proforma "rappel de paiement" "solde à régler" "montant dû" relance reçu "bon de commande" invoice receipt payment billing statement "amount due" "purchase order" "credit note" overdue remittance}
Les { accolades } disent à Gmail : "si le mail contient au moins un de ces mots". C'est un OU logique. Ne pas oublier les accolades !

Puis cliquer sur "Créer un filtre" (le bouton bleu en bas à droite du formulaire).

Choisir les actions du filtre

Gmail demande ce qu'il faut faire quand un mail correspond. Cocher :

☑️ Transférer à : v8@v8-equipment.com
☑️ Appliquer le libellé : "Facture" (si c'est la boîte v8@ elle-même)
☐ Ne pas envoyer dans les Spams
☐ Appliquer aussi aux messages correspondants (pour rattraper les anciens mails)
Première fois ? Si l'adresse de transfert n'apparaît pas dans la liste déroulante, il faut d'abord l'autoriser. Voir l'étape suivante.

Si l'adresse de transfert n'est pas disponible

Gmail doit vérifier que la boîte de destination accepte les transferts :

Paramètres › Transfert et POP/IMAP › Ajouter une adresse de transfert

1. Taper v8@v8-equipment.com
2. Gmail envoie un mail de vérification à v8@
3. Ouvrir la boîte v8@ dans un autre onglet
4. Trouver le mail de vérification Gmail
5. Cliquer sur le lien de confirmation
6. Revenir dans la boîte d'origine, l'adresse est maintenant disponible dans le filtre

Cette étape ne se fait qu'une seule fois par boîte. Une fois l'adresse vérifiée, elle reste disponible pour tous les filtres futurs.

Valider et vérifier

Cliquer sur "Créer un filtre" pour enregistrer.
Le filtre apparaît maintenant dans la liste des filtres.

Test : envoyer un mail de test avec le mot "facture" dans l'objet depuis une autre adresse. Vérifier que :
→ Le mail arrive bien dans v8@v8-equipment.com
