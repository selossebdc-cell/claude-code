# Diagnostic & Config Generator pour clients

**Statut** : Idee brute — a explorer
**Date** : 2026-03-18

## Concept

Productiser le setup Claude Code de Catherine pour ses clients. Une application/agent en deux briques :

### Brique 1 — Diagnostic / Discovery Engine
Parcours conversationnel structure qui interroge le client sur :
- **Mindset & patterns** : mode de fonctionnement, blocages recurrents, style decisionnel, rapport au temps, energie
- **Business** : activite, offre, avatar, process existants, outils utilises, pain points operationnels
- **Habitudes de travail** : rituels, frequence des taches, ce qui prend du temps, ce qui est delegue ou pas

### Brique 2 — Config Generator
A partir du diagnostic, generation automatique de :
- **Skills Claude Code** personnalises (adaptes a leur metier)
- **Agents** avec les bons tools et routines
- **Memory initiale** pre-remplie (profil utilisateur, contexte business)
- **Un "miroir introspectif"** calibre sur leurs patterns (version personnalisee de Noble AI)
