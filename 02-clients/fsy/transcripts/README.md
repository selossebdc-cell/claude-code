# Transcripts - Face Soul Yoga

Ce dossier centralise tous les transcripts de sessions et d'échanges utiles
pour produire des livrables plus précis et personnalisés.

## Règle de nommage

- `YYYY-MM-DD_session-N_source_client.txt`
- Exemple: `2026-04-29_session-04_fathom_fsy.txt`

## Sources acceptées

- Fathom
- Fireflies
- Plaud
- Notes transformées en transcript texte

## Usage

1. Déposer le transcript brut ici.
2. Garder un fichier par session/source.
3. Ajouter dans `SESSIONS.md` le nom du fichier utilisé.
4. Ne pas mettre de secrets/tokens dans les transcripts.
