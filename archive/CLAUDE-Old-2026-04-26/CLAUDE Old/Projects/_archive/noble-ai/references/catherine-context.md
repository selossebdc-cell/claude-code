## Blocages cartographiés

### Procrastination identitaire
Catherine ne procrastine pas par paresse. Sa procrastination est liée à :
- La peur du succès (et de ce qu'il implique)
- Le perfectionnisme comme évitement déguisé
- L'auto-violence ("je devrais déjà avoir fait ça")
- Le brouillard émotionnel qui paralyse la prise de décision

### Boucles récurrentes identifiées
- **Formation** : veut tout perfectionner avant de livrer → ne livre jamais → culpabilise
- **Vente** : prépare, prépare, mais n'envoie pas le message → "c'est pas encore prêt"
- **Prospection** : sait qu'il faut le faire → repousse → se sent nulle → repousse encore
- **Sport** : commence motivée → rate un jour → abandon complet → recommence dans 3 semaines
- **Poids** : restriction → craquage → culpabilité → restriction (boucle classique)
- **Argent/comptes** : évite de regarder → anxiété monte → évite encore plus
- **Multi-projets** : dit oui à tout → surcharge → brouillard → rien n'avance

### Pattern transversal
Le fil rouge : Catherine met l'énergie dans la préparation (zone de confort intellectuel) plutôt que dans l'action visible (zone de risque). La perfection sert de bouclier contre l'exposition.
